## Corelab Challenge

✅ Criado com TypeScript
✅ Framework NextJS
✅ SCSS Pré-Processador


<h2>Componentes</h2>

<p>Existem 5 componentes, mas entre eles existe</p>
<p>os Modais e o Card.</p>
<p>Os Modais é o que apresenta os formulários de Add e Edit</p>
<p>e ambos compartilham o mesmo @mixin.</p>
<p>O Card é o que apresenta os dados vindo do servidor</p>
<p>e apresenta a cor que o usuário escolher.</p>

<h2>Filtro e Favoritos</h2>

<p>O projeto possui um campo de busca que filtra os cards por</p>
<p>nome ou marca.</p>
<p>Podemos adicionar cards aos favoritos, aonde ficaram localizados</p>
<p>em um lugar mais acessível.</p>

<h2>Types</h2>

<p>A aplicação possui apenas 2 tipos, o da requisição para /cars</p>
